Text Generation API

API orientada a la generación de texto original a partir de prompts. Ideal para asistentes virtuales, automatización de contenido, soporte al cliente, copywriting y flujos de trabajo con IA.

TextoChatPromptsInstrucciones

Funcionalidad

Recibe una instrucción en lenguaje natural y devuelve texto coherente, contextual y adaptado al tono solicitado.

Entradas requeridas

{
  "prompt": "Redacta una descripción de producto",
  "language": "es",
  "temperature": 0.7,
  "max_tokens": 120
}

Salida esperada

{
  "generated_text": "Descripción creada por la IA...",
  "usage": {
    "input_tokens": 18,
    "output_tokens": 42
  },
  "request_id": "tg_00192"
}
